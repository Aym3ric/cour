Les deux types de balises

Il existe deux types de balises :

  1. les balises usuelles encadrent du code, par exemple : <p> et </p>
  2. les balises dites orphelines n'encadrent aucun code; elles se ferment elles-mêmes. Par exemple : <br/> abréviations de break row; <hr/> abréviations de horizontal row
